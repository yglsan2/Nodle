Nodle

Ce JAR est lancé en mode logiciel Java mais l’interface n’a pas encore été intégrée.

Pour obtenir l’application complète (web + API dans un seul JAR) :

  1. Construire le frontend : cd frontend && npm install && npm run build
  2. Reconstruire le backend avec le profil : cd backend && mvn package -Pwith-frontend

Puis relancer ce JAR : l’interface sera disponible à la même adresse (ex. /).

Vous pouvez aussi utiliser Nodle en mode web (frontend et backend séparés) ou en application Android (APK) — voir le README.
